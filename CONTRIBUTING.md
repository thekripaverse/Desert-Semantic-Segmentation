# Contributing to DesertVision

Thank you for your interest in improving DesertVision! We welcome contributions that help make desert terrain segmentation more accurate and robust.

## How to Contribute
1. **Report Bugs:** Open an issue describing the bug and how to reproduce it.
2. **Feature Requests:** Suggest new backbones (e.g., EfficientNet) or loss functions via issues.
3. **Pull Requests:** - Fork the repository.
   - Create a new branch (`feat/your-feature`).
   - Ensure your code follows the established docstring and type-hinting style.
   - Run existing tests in the `tests/` folder using `pytest`.
   - Submit a PR with a clear description of changes.

## Development Setup
- Install dependencies: `pip install -r requirements.txt`
- Install dev tools: `pip install pytest black flake8`
- Run tests: `python -m pytest tests/`

## Code of Conduct
Please be respectful and professional in all interactions within this project.
